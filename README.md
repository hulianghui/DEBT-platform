# 杜尔伯特热电联产在线监测平台

基于 macOS 设计风格的电力在线监测平台前端界面。

## 在线预览

🔗 [点击访问](https://hulianghui.github.io/DEBT-platform/macos-dark-ui-pro.html)

## 功能特性

- 🎨 macOS 风格深色主题界面
- 📊 实时数据可视化
- 🔥 燃料监测
- ⚙️ 输煤监测
- ⚡ 生产过程监测
- 💨 环境气体监测
- 🔔 报警中心
- 📦 资产台账
- 📋 工单管理

## 技术栈

- HTML5
- CSS3
- JavaScript
- Chart.js

## 项目结构

```
DEBT-platform/
├── index.html              # 入口文件（自动跳转）
├── macos-dark-ui-pro.html  # 主页面
├── macos-dark-ui-demo.html # 演示页面
├── src/                    # 源代码
└── ...
```

## 本地运行

```bash
# 使用 Python 启动本地服务器
python -m http.server 8080

# 然后访问 http://localhost:8080
```
