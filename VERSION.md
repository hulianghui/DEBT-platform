# 版本记录

## V1.0 (当前版本)
**生成时间**: 2026-03-21
**文件**: macos-dark-ui-pro.html

### 功能模块
1. **燃料监测管理子系统（S1）**
   - 红外热成像实时预览
   - 封闭煤场自燃监测
   - 气体监测（CO/SO₂/CH₄/H₂S）
   - 3D点云智能盘库
   - 温度趋势分析
   - 告警闭环处理

2. **输煤监测管理子系统（S2）**
   - 皮带AI检测（火焰检测、纵撕识别）
   - 设备温度监测
   - 托辊声纹监测
   - 温度曲线分析

3. **生产过程监测子系统（S3）**
   - 汽轮机热成像视图
   - 轴承振动频谱分析
   - 泵/阀/电机温振监测
   - 生产参数趋势

4. **电力设备监测子系统（S4）**
   - 高压开关柜温度监测
   - 变压器温度监测
   - 低压配电柜温度监测
   - RFID无源无线传感器
   - 一次系统图展示

5. **智能预警管理子系统（S5）**
   - 三级预警机制
   - 智能工单管理
   - 工单处理时效统计

6. **资产管理子系统（S6）**
   - 设备健康度分布
   - RUL预测（剩余寿命）
   - 维保计划管理
   - 设备台账

7. **环境气体监测模块**
   - 五种气体监测（CO/SO₂/CH₄/H₂S/O₂）
   - CO浓度趋势图表
   - 多气体对比
   - 监测点列表管理

8. **报表中心**
   - 日报/周报/月报
   - 报告生成统计
   - 报告列表管理

9. **系统管理**
   - 用户管理
   - 角色权限配置
   - 系统日志
   - 数据采集状态
   - 系统设置

### 技术特性
- macOS暗黑风格UI设计
- Tailwind CSS + Chart.js
- 响应式布局
- 实时数据可视化
- 三级预警颜色体系（绿色/黄色/红色）
